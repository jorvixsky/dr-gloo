# Dr Gloo

Dr Gloo is a cross-chain token consolidator.

## Tech Stack

- React
- Tailwind CSS
- Vite
- TypeScript
- RainbowKit
- Wagmi

## Features

It uses CCTPv2 to transfer USDC tokens between chains. Swapping other tokens into USDC is not supported yet.

# Running the app

```bash
bun install
bun run dev
```
